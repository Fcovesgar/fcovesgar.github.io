Padeldist

Descripción

Aplicación móvil para la gestión y reserva de pistas de pádel en un club deportivo, que permite a los usuarios consultar disponibilidad, realizar reservas y consultar eventos del club.

Especificaciones Técnicas

  • Lenguajes: Kotlin, XML, JavaScript
  • Frameworks / SDK: Android SDK, Firebase
  • Herramientas: Android Studio, Git
  • Arquitectura: MVVM (Model-View-ViewModel)
  • Base de datos: Firebase Realtime Database
