TCG Pocket Trader

Descripción

Plataforma web orientada a la comunidad de jugadores de Pokémon TCG Pocket, diseñada para la visualización e intercambio de cartas. Incluye información y filtrado avanzado de las cartas, sistema de intercambios, mensajería en tiempo real, personalización de perfiles y gestión de amigos. Cuenta con un diseño adaptable para todo tipo de dispositivos.

Especificaciones Técnicas

  • Lenguajes: PHP, JavaScript, MySQL, HTML5, CSS, JSX
  • Frameworks: Laravel, React, Tailwind CSS
  • Herramientas: Composer, npm, Git
  • Arquitectura: API REST
  • Servicios externos: Pusher (WebSockets), Cloudinary (almacenamiento en la nube)
